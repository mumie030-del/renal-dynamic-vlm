# 肾动态显像混合性梗阻AI辅助判断指南

## 一、文档说明

### 1.1 用途
本指南用于医生在AI辅助诊断系统输出结果后，进行快速复核与决策。支持临床验证与科研论文撰写。

### 1.2 数据来源
- AI模型：Qwen-Plus (temperature=0.6, top_p=0.95)
- 四分类判断 + 混合型二次验证 + 偏向分析 + 临床辅助
- 文件位置：`dataset/results_temp0.6_topp0.95_mixed_guarded/`

### 1.3 输出格式
- **表格文件**：`results_summary.csv`（Excel/Numverse可打开）
- **详细报告**：每个病例对应一个 `.txt` 文件

## 三、核心判断标准（医生复核用）

### 3.1 四分类定义
| 类型 | TAC特征 | 利尿后表现 |
|------|---------|-----------|
| **机械性梗阻** | 曲线持续上升，无下降段 | 无明显下降或轻微下降（<30%），末帧仍高位滞留 |
| **功能性梗阻** | 曲线有峰有降 | 明显下降（>30%），末帧比值<0.6 |
| **混合性梗阻** | 介于两者之间 | 有下降但末帧比值0.4-0.9 |
| **正常肾脏** | 快速上升后迅速下降 | 利尿后曲线迅速下降，末帧比值<0.3 |

### 3.2 混合性梗阻二次验证规则
AI会触发二次验证的情况（`is_mixed_candidate`）：

| 条件 | 触发标准 |
|------|---------|
| **条件1** | 首次判断已是混合型 → 必须验证 |
| **条件2** | 轻中度下降 + end_ratio ∈ [0.45, 0.90] → 验证 |
| **条件3** | 明显下降 + 滞留偏高 + end_ratio ∈ [0.55, 0.80] → 验证 |
| **条件4** | 不下降 + 滞留非高位 + end_ratio ∈ [0.55, 0.80] → 验证 |

### 3.3 混合型偏向分析评分标准
AI给出偏向评分（1-5分）：
- **偏功能性**：利尿后明显下降，排空改善明显（评分 1-2）
- **偏机械性**：利尿后下降不明显，仍高��滞留（评分 4-5）
- **均衡型**：介于两者之间（评分 3）

## 四、医生复核清单

### 4.1 基础分类复核
- [ ] 右肾类型判断是否合理？（对照TAC描述）
- [ ] 左肾类型判断是否合理？
- [ ] 置信度为"中"或"低"时需重点复核推理过程

### 4.2 混合型病例复核（重点）
当任一肾脏为"混合性梗阻"时：

**步骤1：检查二次验证结论**
```
查看字段：右肾验证类型 / 左肾验证类型
期望值：
  - "保持原判断" → 置信度高则接受，低则质疑
  - "混合性梗阻" → 确认TAC数据是否支持
```

**步骤2：核对TAC数据支撑**
```
右肾/左肾TAC：
  峰值帧：正常在2-6帧（2-3min）
  利尿后趋势：
    - "明显下降" → 支持功能性因素
    - "轻微下降" → 支持机械性因素
    - "无下降" → 纯机械性
  末帧比值：
    - <0.4 → 正常或功能性
    - 0.4-0.6 → 轻度混合
    - 0.6-0.8 → 中度混合
    - >0.8 → 重度混合/机械性
  滞留描述：
    - "低位滞留" → 功能性为主
    - "中度滞留" → 混合性
    - "高位滞留" → 机械性为主
```

**步骤3：判断偏向分析是否合理**
```
偏向评分与TAC一致性检查：
  偏向"偏功能" + 利尿后明显下降 + 末帧比值0.4-0.6 → ✓ 合理
  偏向"偏机械" + 利尿后无下降 + 末帧比值>0.7 → ✓ 合理
  偏向"均衡" + 末帧比值0.55-0.75 → ✓ 合理
  
  不一致时需标注：★ 标记为"待讨论"
```

### 4.3 临床辅助判断复核
- [ ] 诊断要点是否与AI分类一致？
- [ ] 检查建议是否合理？（CTU/MRU/超声/尿动力学）
- [ ] 治疗优先级判断是否正确？（紧急/限期/保守）
- [ ] 随访建议是否恰当？（3m/6m/12m）

## 九、使用建议

1. **每日工作流**：
   - 打开 `results_summary.csv` 查看待复核列表
   - 双击病例名打开对应 `.txt` 文件查看详情
   - 在疑难病例记录表中标注

2. **撰写论文**：
   - 从表格筛选目标病例（如：所有混合型）
   - 统计AI判断与医生复核的一致性
   - 引用TAC数据支撑论点

3. **质量控制**：
   - 置信度为"低"的病例100%复核
   - 混合型病例必须查看二次验证和偏向分析
   - 标注所有"△"和"★"病例，组织讨论
